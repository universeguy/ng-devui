## toggle 说明

| 参数        | 类型          | 默认        |   说明                 |
| :---------: | :----------: | :---------: | :------------------------------------------|
| size        | string       | medium      | 大小，例如 small，medium，big |
| color       | string       | #5170FF     | 开关打开时的颜色 |
| checked       | boolean       | false     | 开关打开 |
| change       | func       | --     | 切换状态回调 |
