| 参数        | 类型          | 默认        |   说明                 |
| :---------: | :----------: | :---------: | :------------------------------------------|
| type        | string       | 'info'      | 必选参数，指定警告提示的样式，有四种选择 success\|danger\|warning\|info |
| closeable   | boolean      | true        | 可选参数，默认显示关闭按钮 |
| closeEvent     | function     | (none)      | 可选参数，关闭时触发的回调函数 |
| dismissTime | number       | (none)      | 可选参数，自动关闭alert的延迟时间(单位：ms) |
