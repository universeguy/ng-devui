| 参数            | 类型          | 默认        |   说明                 |
| :---------:     | :----------: | :---------: | :------------------------------------------|
| percentage      | number       | 0           | 进度条的值最大为100 |
| percentageText  | string       | (none)      | 进度条当前值的文字说明比如：'30%' \| '4/5' |
| barbgcolor      | string       | #5170ff     | 进度条的颜色显示，默认为天蓝色 |
| height          | string       | 20px        | 进度条的高度值，默认值为20px |
